goBasic()

Launches the basic view of the calcuator, which is the default view for this calculator. Here we can do basic arithmetic such as add, subtract, multiply, and divide. We can also do more slighly complicated functions such as squaring, inversing, or taking a square root. Values and operations can be input via key presses or mouse clicks, and once the user presses the equals button, the result is displayed on the screen above.
